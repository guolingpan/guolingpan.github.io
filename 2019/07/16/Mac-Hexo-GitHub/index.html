Mac+Hexo+GitHub

Posted on 2019-07-16

1.安装nodejs和git

https://nodejs.org/en/

  • node -v
  • npm -v
  • Git –version

2.安装Hexo

sudo npm install –unsafe-perm=true –allow-root –save-dev grunt -g hexo
–unsafe-perm=true –allow-root 权限控制
-g全局

3.博客初始化

  1. 新建blog文件夹
    cd blog
  2. 执行下述命令初始化本地博客，下载一系列文件。
    sudo hexo init –unsafe-perm=true –allow-root
  3. 执行下述命令安装npm。
    sudo npm install –unsafe-perm=true –allow-root
    sudo npm install –unsafe-perm=true –allow-root
  4. 生成本地网页文件并开启服务器，然后通过http://localhost:4000查看本地博客。
    sudo hexo g
    sudo hexo s

4.本地博客代码上传GitHub

  1. 注册并登陆GitHub账号后，新建仓库，名称必须为user.github.io，如guolingpan.github.io。
  2. 修改vim _config.yml文件：
    deploy:
    type: git
    repository: https://github.com/guolingpan/guolingpan.github.io.git
    branch: master
    注意type、repository、branch后均有空格。
  3. 生成静态文件并上传到服务器
    sudo hexo g （npm install hexo –save）
    sudo hexo d （npm install hexo-deployer-git –save）
    可通过https://guolingpan.github.io和http://localhost:4000访问

5.添加ssh keys到GitHub

ssh-keygen -t rsa -C “your email”
open ~/.ssh打开文件夹，复制id_rsa.pub 内容到github上的setings中的 ssh keys中
